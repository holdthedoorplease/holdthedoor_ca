Immigration to Canada#

Express Entry#

Express Entry is a fast-track immigration system used for skilled workers, professionals, and international students.

Federal Skilled Worker (EE)

Apply through the Federal Skilled Worker program for fast-tracked immigration.

https://www.canada.ca/en/immigration-refugees-citizenship/services/immigrate-canada/express-entry/works.html

Canadian Experience Class (EE)

Utilize the Canadian Experience Class for those with Canadian work experience.

https://www.canada.ca/en/immigration-refugees-citizenship/services/immigrate-canada/express-entry/experience.html

Federal Skilled Trades (EE)

Apply through the Federal Skilled Trades Program for skilled workers in specific trades.

https://www.canada.ca/en/immigration-refugees-citizenship/services/immigrate-canada.html

Provincial Nominee Program#

The Provincial Nominee Program (PNP) is a program that allows Canadian provinces and territories to nominate skilled workers for permanent residence.

Learn about the British Columbia Provincial Nominee Program for skilled workers.
